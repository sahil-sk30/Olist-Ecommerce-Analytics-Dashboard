# Olist E-Commerce Sales & Customer Analytics Dashboard

## Project Overview
This project analyzes 100,000+ e-commerce orders to evaluate sales performance, customer behavior, delivery efficiency, and customer satisfaction.

## Tools Used
- Excel
- Power Query
- Pivot Tables
- Power BI
- SQL

## Key Insights
- Analyzed 100K+ e-commerce orders
- Identified top-performing product categories
- Evaluated delivery performance and shipping delays
- Analyzed customer reviews and satisfaction trends
- Built KPI dashboards for business reporting

## Dashboard Features
- Sales Performance Analysis
- Customer Insights
- Delivery Performance Tracking
- Revenue & Profit KPIs
- Interactive Dashboard Visualizations
